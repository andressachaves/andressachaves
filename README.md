## 🙋‍♀️ Sobre mim

Graduada em **Tecnologia em Banco de Dados pela PUC Minas Gerais**, atualmente atuando como **Monitoring Analyst** na Quero Passagem Viagens e Turismo.

Paralelamente, participo de um projeto de pesquisa científica no **[Lecar — Reef Systems Ecology and Conservation Lab (UFF)](https://lecar.uff.br/)**, orientada por Cientistas de Dados. No projeto, aplico **Visão Computacional, Deep Learning e IA** para contribuir com a preservação e monitoramento de ambientes recifais, com foco em **bioacústica marinha**.

Também desenvolvo um **chatbot RAG com LangChain e MongoDB Atlas** para melhorar a busca de artigos e publicações científicas do laboratório.

- 🔭 Pesquisadora em **IA aplicada à conservação marinha** — Lecar/UFF
- 💼 **Monitoring Analyst** → transição para Analista de Dados
- 🗄️ Apaixonada por **Banco de Dados, SQL, Python e ETL**
- 🤖 Construindo soluções com **LLMs, RAG e Redes Neurais (CNN)**
- ☁️ Explorando **Cloud Computing, Big Data e MLOps**
